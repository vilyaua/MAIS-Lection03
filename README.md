# MAIS-Lection03

## Homework: Research Agent

An interactive AI agent that takes a user's question, autonomously searches the web, reads articles, and generates a structured Markdown report.

### Project Structure

- **`homework-lesson-3/`** — Original homework skeleton with task description
- **`research-agent/`** — Completed implementation
- **`AGENTS.md`** — Implementation guide and specifications

### Quick Start

```bash
cd research-agent
pip install -r requirements.txt
cp .env.example .env   # add your API key
python main.py
```

See [`research-agent/README.md`](research-agent/README.md) for full setup and architecture details.
